## 刷题策略
- **每日目标**：1-2道中等题 或 1道困难题
- **重点类型**：滑动窗口、二叉树、动态规划、链表
- **复习频率**：每周日复习本周错题

## 刷题模板

每道题记录以下信息：
```
### [题目编号]. [题目名称](https://leetcode.cn/problems/xxx/)
- 难度：Easy/Medium/Hard
- 类型：滑动窗口/双指针/DP/DFS/BFS/...
- 日期：YYYY-MM-DD
- 用时：XX分钟
- 状态：✅ AC / ❌ 需复习 / ⏳ 未完成
- 思路：简述解题思路
- 代码：关键代码片段
```
